Algorithm Framework
01
Linear & Logistic Regression
LR — Regression / Classification
Supervised

Fits a line (or decision boundary) through labeled data points by minimizing prediction error. Logistic regression uses a sigmoid function to output probabilities for binary classification. The simplest and most interpretable models in ML.

Tabular
Real-world use cases
House price prediction from square footage
Email spam detection (logistic)
Credit default probability scoring
Complexity
Low
02
Decision Trees
DT — Classification & Regression
Supervised

Recursively partitions data by asking binary questions at each node (e.g., "Is age > 30?"), forming a flowchart-like tree. Each leaf represents a decision outcome. Highly interpretable but prone to overfitting on deep trees.

Tabular
Real-world use cases
Medical diagnosis rule sets
Loan approval decisions
Customer churn segmentation
Complexity
Low
03
Random Forest
RF — Ensemble Method
Supervised

Builds hundreds of decision trees on random data subsets and feature subsets, then aggregates their outputs by majority vote or averaging. Dramatically reduces overfitting compared to a single tree. A go-to baseline for tabular data.

Tabular
Real-world use cases
Fraud detection in banking transactions
Stock market movement classification
Disease diagnosis from clinical data
Complexity
Mid
04
Gradient Boosting
XGBoost / LightGBM / CatBoost
Supervised

Builds weak learners (shallow trees) sequentially — each new tree corrects the errors of the previous one, following the gradient of the loss function. XGBoost consistently dominates structured data competitions (Kaggle).

Tabular
Real-world use cases
Search ranking at tech companies
Click-through rate prediction in ads
Insurance risk modeling
Complexity
Mid
05
Support Vector Machines
SVM — Classification & Regression
Supervised

Finds the hyperplane that maximizes the margin between two classes. Uses the "kernel trick" to project data into higher-dimensional spaces where it becomes linearly separable. Highly effective on small datasets with clear margins.

Tabular NLP
Real-world use cases
Handwritten digit recognition
Text classification (pre-transformer)
Bioinformatics protein classification
Complexity
Mid
06
K-Means Clustering
KM — Clustering
Unsupervised

Partitions unlabeled data into K clusters by iteratively assigning each point to its nearest centroid and recomputing centroids. Converges when assignments stop changing. Sensitive to initial centroid placement and number of K.

Tabular
Real-world use cases
Customer segmentation for marketing
Image color quantization (compression)
Document topic grouping
Complexity
Low
07
Principal Component Analysis
PCA — Dimensionality Reduction
Unsupervised

Identifies directions (principal components) of maximum variance in high-dimensional data and projects data onto fewer dimensions. Compresses information while retaining the most meaningful structure. Uses linear algebra (SVD).

Tabular Vision
Real-world use cases
Visualizing high-dimensional genomics data
Noise reduction in sensor data
Feature compression before supervised ML
Complexity
Low
08
Convolutional Neural Network
CNN — Image Understanding
Deep Learning

Uses convolutional filters that slide over image grids to detect local features (edges → textures → objects). Pooling layers reduce spatial dimensions. Stacking convolutional layers allows hierarchical feature learning from pixels to concepts.

Computer Vision Tabular
Real-world use cases
Medical image diagnosis (X-ray, MRI)
Self-driving car object detection
Face recognition and verification
Complexity
High
09
Transformer (BERT / GPT)
Attention-based — NLP & Vision
Deep Learning

Uses self-attention to weigh the relevance of every token to every other token in a sequence simultaneously — enabling long-range dependencies without recurrence. BERT learns bidirectional context; GPT uses causal (left-to-right) attention for generation.

NLP Vision Gen AI
Real-world use cases
Sentiment analysis & text classification
Machine translation (DeepL, Google)
Conversational AI assistants
Complexity
Very High
10
Diffusion Models
DDPM / Stable Diffusion / DALL·E
Generative AI

Learns to reverse a gradual noise-adding (diffusion) process: during training, Gaussian noise is added step-by-step to images; the model learns to denoise. At inference, it starts from pure noise and denoises iteratively toward a coherent output guided by a text prompt.

Vision Gen AI
Real-world use cases
Text-to-image synthesis (Midjourney, DALL·E)
Drug molecule design & discovery
Video and 3D asset generation
Complexity
Very High
